# 🎵 Music Player

A modern and responsive Music Player built using HTML, CSS, and JavaScript. This project provides an interactive music-playing experience with playback controls, progress tracking, volume adjustment, and a clean user interface.

## 🚀 Features

- ▶️ Play and Pause Music
- ⏮️ Previous Track Navigation
- ⏭️ Next Track Navigation
- 📊 Interactive Progress Bar
- 🔊 Volume Control
- ⏱️ Current Time and Duration Display
- 🎨 Responsive and Modern UI Design
- 💻 Built with Pure HTML, CSS, and JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
